用 not-book
管理你的任务

not-book 是一个基于 Django 构建的任务管理应用，结构清晰、易于扩展， 帮助你高效地组织与追踪待办事项。

进入管理后台 → 查看源码

核心特性

简洁的项目结构，开箱即用

📋

任务应用

内置 task 应用，提供模型、视图与管理后台的标准化骨架，便于快速迭代。

🛠️

Django Admin

开箱即用的管理后台，方便对数据进行增删改查与运维管理。

🗄️

SQLite 数据库

默认集成 SQLite，零配置即可运行，也支持切换到 PostgreSQL / MySQL。

⚙️

ASGI / WSGI

同时提供 ASGI 与 WSGI 入口，支持同步与异步部署场景。

🔒

安全中间件

内置 CSRF、点击劫持防护、会话与认证等安全中间件链。

🌱

易于扩展

清晰的应用分层与迁移机制，方便持续添加新功能与数据模型。

技术栈

基于成熟稳定的 Python 生态
