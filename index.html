not-book
Vue 3 + Django · 全栈架构

用 not-book
轻松管理你的任务

not-book 是一个前后端分离的任务管理系统，前端基于 Vue 3 + Element Plus，后端基于 Django， 支持多角色权限管理，让你的工作更高效、更有序。

开始使用 了解更多

核心功能

简洁高效，开箱即用

任务管理

支持任务的创建、编辑、删除，按状态和优先级分类管理，一目了然。

角色权限

三级角色体系：超级管理员、管理员、普通用户，精细化权限控制。

数据统计

直观的仪表盘统计面板，实时掌握任务进度与团队工作状态。

响应式设计

完美适配桌面端和移动端，随时随地管理你的任务。

API 友好

前后端分离架构，RESTful API 设计，易于扩展和集成。

开源免费

项目完全开源，代码结构清晰，便于学习和二次开发。

技术栈

前后端分离，成熟稳定的技术选型
