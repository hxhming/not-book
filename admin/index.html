项目简介

not-book 是一个轻量级的任务管理系统，采用前后端分离架构设计。前端基于 Vue 3 + Element Plus 构建，后端计划采用 Django + Django REST Framework 实现。

核心特性

  • 任务管理：支持任务的增删改查，按状态（待办/进行中/已完成）和优先级（高/中/低）分类
  • 用户管理：三级角色体系（超级管理员/管理员/普通用户），灵活的权限控制
  • 数据持久化：前端基于 localStorage 实现数据本地存储，可无缝对接后端 API
  • 响应式设计：支持桌面端和移动端访问，自适应布局
  • 仪表盘：直观的数据统计面板，一目了然掌握任务进度

数据版本控制

系统使用 USER_DATA_VERSION 常量控制数据结构版本。当数据结构变更时，递增版本号，用户首次访问时会自动重置为新的种子数据，确保数据一致性。

const USER_DATA_VERSION = 2;

function loadUsers() {
  const data = localStorage.getItem('users');
  const version = parseInt(localStorage.getItem('users_version') || '0', 10);
  if (!data || version < USER_DATA_VERSION) {
    // 重置为新种子数据
    const seed = [...];
    localStorage.setItem('users', JSON.stringify(seed));
    localStorage.setItem('users_version', String(USER_DATA_VERSION));
    return seed;
  }
  return JSON.parse(data);
}
